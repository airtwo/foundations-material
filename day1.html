What does HTML do?

HTML acts as the skeleton of a web page, providing its underlying structure. You can use HTML to specify parts of your page like:

  • header and footer
  • sections
  • text headings
  • body text
  • images
  • links
  • form inputs and buttons
  • items in a list

and more. HTML is also used to indicate what elements of a web page are logically enclosed in other elements. For example, a button that says "Submit Feedback" might belong inside a form, which is inside the same section as the heading, "Website Feedback".
